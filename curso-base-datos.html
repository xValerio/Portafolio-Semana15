Curso de Base de Datos II

Impartido por Raúl Enrique Fernández Bejarano

Universidad Peruana Los Andes

Introducción

Este curso está diseñado para proporcionar a los estudiantes los conocimientos y habilidades necesarios para comprender y desarrollar bases de datos utilizando técnicas avanzadas. A lo largo del curso, se abordarán temas como la normalización, el diseño de bases de datos, el lenguaje SQL y la optimización de consultas, con un enfoque práctico que permita a los estudiantes aplicar lo aprendido en proyectos reales.

Objetivos del Curso

  • Comprender los principios fundamentales de las bases de datos relacionales.
  • Aprender a diseñar y normalizar bases de datos eficientes.
  • Dominar el lenguaje SQL para crear, modificar y consultar bases de datos.
  • Desarrollar habilidades en la optimización de consultas para mejorar el rendimiento.
  • Aplicar conceptos de bases de datos en proyectos reales y trabajos prácticos.

Temario

El curso está dividido en varios módulos que cubren los siguientes temas:

  1. Introducción a las bases de datos relacionales.
  2. Modelado de datos y normalización.
  3. Lenguaje SQL: creación y manipulación de bases de datos.
  4. Consultas complejas en SQL.
  5. Optimización de bases de datos y rendimiento de consultas.
  6. Proyecto final: desarrollo de una base de datos para una empresa.

Metodología

El curso se lleva a cabo a través de clases teóricas y prácticas, en donde los estudiantes tendrán la oportunidad de trabajar con herramientas y tecnologías actuales para diseñar y gestionar bases de datos. Las clases estarán acompañadas de ejercicios prácticos y un proyecto final que permitirá a los estudiantes aplicar lo aprendido en un contexto real.

Requisitos

Para tomar este curso, se recomienda que los estudiantes tengan conocimientos previos en programación y en el uso de sistemas de gestión de bases de datos (DBMS). Se requiere un nivel básico de conocimiento en SQL y estructuras de datos.

Evaluación

La evaluación del curso se basará en:

  • Exámenes parciales (30%)
  • Trabajos prácticos (40%)
  • Proyecto final (30%)
